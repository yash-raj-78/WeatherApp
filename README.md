🌦️ Weather AppA sleek and intuitive Android application built with Kotlin that provides real-time weather updates for any location worldwide.

🚀 Features
Real-time Updates: Fetches live temperature, humidity, and atmospheric conditions.
Global Search: Allows users to search for any city to get instant weather data.
Intuitive UI: Clean and user-friendly interface designed for efficient weather tracking.
Secure Integration: Utilizes REST APIs to ensure reliable and secure data fetching.

🛠️ Tech Stack
Language: Kotlin 
Networking: REST API 
Tools: Android Studio
